## 🌌 Une page se tourne

Le développement continue désormais sous le nom **Stelliverse**. Ce dépôt GitHub reste en ligne pour l'**historique** et les [**anciennes releases**](https://github.com/drrakendu78/TradSC/releases), mais **n'est plus maintenu**.

### ➡️ Tout se passe maintenant sur **[stelliverse.fr](https://stelliverse.fr/)**

- 🆕 **Stelliverse** reprend tout ce que faisait StarTrad — traduction FR, overlay en jeu, carnet de bord, outils pilote, compagnon mobile — en **mieux**, et continue d'évoluer.
- 🔁 **Rien n'est perdu** : ton compte et tes sauvegardes sont conservés. Reconnecte-toi avec le **même Discord** et tout est là.
- 📥 La **dernière version de StarTrad (4.4.0)** bascule automatiquement vers Stelliverse.
