# StarTrad FR - Version 3.0.5

## Nouveautés

### Détection et lancement du RSI Launcher
- Nouveau bouton "Lancer le Launcher" sur la page d'accueil si le RSI Launcher est détecté
- Détection automatique du launcher via les logs RSI (même méthode que la détection du jeu)
- Fallback sur les chemins standards et le registre Windows si non trouvé dans les logs
- Si le launcher n'est pas installé, le bouton "Télécharger le Launcher" reste affiché
- Lancement en processus détaché pour ne pas bloquer l'application
