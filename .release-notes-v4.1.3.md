## ✨ Nouveautés

**Nettoyage automatique des caches obsolètes** (désactivé par défaut)

Activez l'option dans **Paramètres → Services** pour que l'app supprime automatiquement les caches Star Citizen qui ne correspondent plus à une version installée. Par exemple : après un patch de 4.7.0 vers 4.8.0, le cache `sc-alpha-4.7.0` est supprimé automatiquement pour libérer de l'espace disque.

**Logique safe** : seuls les caches dont la version n'est plus installée nulle part (LIVE, PTU, EPTU, TECH-PREVIEW) sont supprimés. Aucun risque de toucher au cache d'une version courante.

**Déclencheurs** (uniquement si l'option est activée) :
- Au démarrage de l'app
- Avant chaque ouverture du RSI Launcher

Toast de confirmation à chaque nettoyage avec le nombre de Mo libérés.

L'option est aussi proposée dans le **wizard d'onboarding** (étape Services) pour les nouveaux utilisateurs, et synchronisée avec celle des Paramètres.
