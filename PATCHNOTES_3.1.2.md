# StarTrad FR - Version 3.1.2

## Mises à jour automatiques

### Suppression automatique de l'installeur
- L'installeur (.exe) est maintenant supprimé automatiquement du dossier Téléchargements après l'installation
- Plus besoin de nettoyer manuellement les anciens fichiers d'installation

### Correction de la double installation
- Correction d'un bug où cliquer sur "Installer maintenant" relançait une deuxième installation
- Utilisation d'un système de signal (fichier flag) pour communiquer avec le script d'installation
- Une seule installation, un seul redémarrage

## Corrections

### Synchronisation Cloud
- Correction du bouton "Écraser" qui ne fonctionnait pas dans les préférences cloud
- Amélioration de l'affichage du nombre de sauvegardes

### Page d'accueil
- Correction du clic sur les actualités qui n'ouvrait pas les liens dans le navigateur
- Les actualités s'ouvrent maintenant correctement comme sur la page Actualités
