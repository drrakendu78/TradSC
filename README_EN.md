## 🌌 Turning the page

Development now continues under the name **Stelliverse**. This GitHub repository stays online for **history** and [**past releases**](https://github.com/drrakendu78/TradSC/releases), but is **no longer maintained**.

### ➡️ Everything now happens at **[stelliverse.fr](https://stelliverse.fr/)**

- 🆕 **Stelliverse** does everything StarTrad did — FR translation, in-game overlay, logbook, pilot tools, mobile companion — only better, and keeps evolving.
- 🔁 **Nothing is lost**: your account and saves are kept. Sign back in with the **same Discord** and everything is there.
- 📥 The **last StarTrad version (4.4.0)** automatically migrates you to Stelliverse.
